Continuity and differentiability

A function is continuous at a point if it is where you expect it to be at that point. A function is differentiable at a point if it's locally linear there.

A function is continuous on a set if it's continuous at every point in that set, and likewise with differentiability.

Both of these notions can be expressed rigorously.

Continuity

A function f:Rm→Rnf: \mathbb{R}^m \to \mathbb{R}^nf:Rm→Rn is continuous at a point ccc iff the image of every sequence converging to ccc converges to f(c)f(c)f(c). That is, for all sequences σ:N→Rm\sigma : \mathbb{N} \to \mathbb{R}^mσ:N→Rm for which σ→c\sigma \to cσ→c, we have f∘σ→f(c)f \circ \sigma \to f(c)f∘σ→f(c).

Referring to our slogan above, each sequence σ→c\sigma \to cσ→c establishes an “expectation”:

  1. Select any sequence σ\sigmaσ that converges to ccc.

  2. Obscure f(c)f(c)f(c) so we can't see its value.

  3. Now “track” f∘σf \circ \sigmaf∘σ: as σ→c\sigma \to cσ→c, is f∘σf \circ \sigmaf∘σ converging to a point in the codomain? If so, this is where we “expect” f(c)f(c)f(c) to be.

A function is then continuous at ccc if all of these possible “expectations” are consistent.

Differentiability

A function f:Rm→Rnf: \mathbb{R}^m \to \mathbb{R}^nf:Rm→Rn is (totally) differentiable at a point ccc iff there exist L,ρ:Rm→RnL, \rho : \mathbb{R}^m \to \mathbb{R}^nL,ρ:Rm→Rn such that

f(x)=f(c)+L(x−c)+ρ(x−c)f(x) = f(c) + L(x - c) + \rho(x - c)f(x)=f(c)+L(x−c)+ρ(x−c)

where LLL is a linear transform and ρ\rhoρ vanishes as we “zoom in”:

In this case we're justified in calling the transform LLL the derivative of fff at ccc. The fact that ρ\rhoρ vanishes as we zoom in on the graph of fff at ccc is what we mean by “locally linear”. But why this strange form? This is simply a generalization of the one-dimensional case, where a differentiable function fff may be expressed as

f(x)=f(c)+f′(c)(x−c)+ρ(c)f(x) = f(c) + f'(c)(x - c) + \rho(c)f(x)=f(c)+f′(c)(x−c)+ρ(c)

If we define L(x)=f′(c)xL(x) = f'(c)xL(x)=f′(c)x1 then we can rewrite this as:

f(x)=f(c)+L(x−c)+ρ(c)f(x) = f(c) + L(x - c) + \rho(c)f(x)=f(c)+L(x−c)+ρ(c)

In higher dimensions, linear transforms may do more than just “stretch” the space: they might skew it, rotate it, mirror it, etc. However, linear transforms are still much more constrained than general functions.
