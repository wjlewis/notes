Topology

A topological space is a set XXX along with a notion of what it means for a set to be “open” in XXX. Specifically, a topogical space is a pair ⟨X,T⟩\langle X, \mathcal{T} \rangle⟨X,T⟩, where T⊆2X\mathcal{T} \subseteq 2^XT⊆2X satisfying 3 properties:

  1. ∅∈T\emptyset \in \mathcal{T}∅∈T and X∈TX \in \mathcal{T}X∈T

  2. T\mathcal{T}T is closed under finite intersections

  3. T\mathcal{T}T is closed under arbitrary unions

In this case T\mathcal{T}T is called a topology on XXX.

Examples

Given any set XXX, the trivial topology on XXX contains only the empty set and XXX itself: ⟨X,{∅,X}⟩\langle X, \left\{\emptyset, X\right\} \rangle⟨X,{∅,X}⟩. This clearly satisfies property 1, and it's easy to check the other two.

At the other extreme lies the discrete topology on XXX, which contains every subset: ⟨X,2X⟩\langle X, 2^X \rangle⟨X,2X⟩. Again, this clearly satisfies the conditions above.

It's often possible to define many different topologies on a single given set. For instance, R\mathbb{R}R admits the familiar topology:

⟨R,{S⊆R∣x∈S  ⟹  Bε(x)⊆S}⟩\langle \mathbb{R}, \left\{ S \subseteq \mathbb{R} \mid x \in S \implies B_\varepsilon(x) \subseteq S \right\} \rangle⟨R,{S⊆R∣x∈S⟹Bε​(x)⊆S}⟩

Additional concepts

Because topological spaces are so general, we forfeit many of the “nice” properties of, say, R\mathbb{R}R. However, we can still formulate familiar notions like “inside”, “outside”, “on the boundary”, etc.

Given a topological space ⟨X,T⟩\langle X, \mathcal{T} \rangle⟨X,T⟩, a set S⊆XS \subseteq XS⊆X (note that SSS is not necessarily a subset of T\mathcal{T}T), and a point p∈Xp \in Xp∈X,

  • ppp is in the interior of SSS iff there's an open set containing ppp that is itself completely contained in SSS: that is, if there exists U∈TU \in \mathcal{T}U∈T such that p∈Up \in Up∈U and U⊆SU \subseteq SU⊆S.

  • ppp is in the exterior of SSS iff it's in the interior of ScS^cSc.

  • ppp is on the boundary of SSS iff it's neither in the interior or exterior of SSS.

With these definitions in hand we can define the interior, exterior, and boundary of a set in the obvious way:

  • The interior of SSS, denoted S∘S^\circS∘, is the set of interior points of SSS.

  • The exterior of SSS, denoted Ext(S)\mathrm{Ext}(S)Ext(S), is the set of exterior points of SSS.

  • The boundary of SSS, denoted ∂S\partial S∂S, is the set of boundary points of SSS.

These allow us to define the closure of a set:

  • The closure of SSS, denoted S‾\overline{S}S, is the union of SSS with its boundary: S∪∂SS \cup \partial SS∪∂S.

Finally, we define accumulation points of a set:

  • ppp is an accumulation point of SSS iff every open set containing ppp overlaps SSS at some point other than ppp. That is, for all U∈TU \in \mathcal{T}U∈T, if p∈Up \in Up∈U, then (U−{p})∩S≠∅\left(U - \{p\}\right) \cap S \ne \emptyset(U−{p})∩S=∅. Accumulation points are arbitrarily “near” to other points in SSS in the sense that any neighborhood containing ppp also contains some other point from SSS.

Note that an accumulation point of a set does not need to be an element of that set. For instance, in R\mathbb{R}R with the usual topology, 000 is an accumulation point of (0,1)(0, 1)(0,1), even though 0∉(0,1)0 \not\in (0, 1)0∈(0,1).

Things are even stranger in the right-ray topology: suppose S∈RS \in \mathbb{R}S∈R is nonempty and bounded above; then ppp is an accumulation point of SSS iff p≤sup(S)p \le \mathrm{sup}(S)p≤sup(S). (If SSS is empty, then it has no accumulation points; if SSS is not bounded above, then every point in R\mathbb{R}R is an accumulation point).
