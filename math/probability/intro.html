Probability

What are the odds that we roll a 2, 4, or 6 with a standard 6-faced die? This is the kind of question we're trying to answer. We say that the probability of this occurring is P(2,4,6)=12P({2, 4, 6}) = \frac{1}{2}P(2,4,6)=21​. But what does this mean? Specifically, what does the 12\frac{1}{2}21​ mean here? After all, it's possible to roll a die many times and not see a 2, 4, or 6. Instead, the probability should reflect the long-term behavior of a system. We'd like P(2,4,6)=12P({2, 4, 6}) = \frac{1}{2}P(2,4,6)=21​ to mean that if we repeatedly roll a die, counting the number of times we rolled an even value, then the ratio of this count to the total number of rolls should converge to 12\frac{1}{2}21​. So already we see that any theory of probability touches concepts like measurement and convergence.

Big question: how do we handle (or sidestep) the idea that “nature” is deciding an outcome in our experiments? This feels impossible to encode mathematically, so the solution should be pretty clever.

Basic concepts

A sample space is a collection of “things that can happen”: the set of all possible outcomes. For instance, the sample space for our dice roll scenario might be {1,2,3,4,5,6}\{1, 2, 3, 4, 5, 6\}{1,2,3,4,5,6}, the sample space for a coin toss experiment could be {0,1}\{0, 1\}{0,1}, etc. The sample space is typically denoted by Ω\OmegaΩ.

An event space F\mathcal{F}F is a σ\sigmaσ-algebra on Ω\OmegaΩ. That is, an event is a set of outcomes from the sample space. For instance, {2,4,6}\{2, 4, 6\}{2,4,6} is the event discussed in the introduction. We think of an event as having “occurred” after an experiment if the “outcome” of the experiment is an element of the event. For instance, the event {2,4,6}\{2, 4, 6\}{2,4,6} occurs if we roll a 444, but not if we roll a 333.

A probability measure P:F→[0,1]P : \mathcal{F} \to [0, 1]P:F→[0,1]. That is, it maps each event to a probability, a value in [0,1][0, 1][0,1]. As indicated by the name, PPP must be a measure with some additional constraints:

  • P(∅)=0P(\emptyset) = 0P(∅)=0 and P(Ω)=1P(\Omega) = 1P(Ω)=1

  • P(⋃i=0∞Ai)=∑i=0∞P(Ai)P\left(\bigcup_{i = 0}^\infty A_i\right) = \sum_{i = 0}^\infty P(A_i)P(⋃i=0∞​Ai​)=∑i=0∞​P(Ai​) for pairwise disjoint events {Ai∣i∈N}\{ A_i \mid i \in \mathbb{N} \}{Ai​∣i∈N}.

The second of these is called “countable addivity” or “σ\sigmaσ-additivity”.

Together, the triple ⟨Ω,F,P⟩\langle \Omega, \mathcal{F}, P \rangle⟨Ω,F,P⟩ is called a probability space.

Remark: Some subsets of Ω\OmegaΩ might be beyond our consideration. That is, we might be able to think of an event, but not be able to determine its probability, because it doesn't “fit” into any σ\sigmaσ-algebra on the sample space.

Basic properties

Not surprisingly, the probability of the complement of any event is 111 minus the probability of the original event. This is a direct consequence of the fact that P(Ω)=1P(\Omega) = 1P(Ω)=1. In particular, 1=P(Ω)=P(A∪Ac)=P(A)+P(Ac)1 = P(\Omega) = P(A \cup A^c) = P(A) + P(A^c)1=P(Ω)=P(A∪Ac)=P(A)+P(Ac), so P(Ac)=1−P(A)P(A^c) = 1 - P(A)P(Ac)=1−P(A).
