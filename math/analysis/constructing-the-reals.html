Contructing the reals

In this note we'll work through Cantor's construction of the real numbers.

Cauchy sequences of rationals

We first define a set RRR containing all Cauchy sequences of rational numbers:

R={σ:N→Q∣σ is Cauchy}R = \left\{ \sigma : \mathbb{N} \to \mathbb{Q} \mid \sigma \text{ is Cauchy} \right\}R={σ:N→Q∣σ is Cauchy}

where a sequence of rationals is Cauchy1 iff

∀B∈N+,∃N∈N:m,n≥N  ⟹  ∣σ(m)−σ(n)∣<1B\forall B \in \mathbb{N}^+, \exists N \in \mathbb{N} : m, n \ge N \implies |\sigma(m) - \sigma(n)| < \frac{1}{B}∀B∈N+,∃N∈N:m,n≥N⟹∣σ(m)−σ(n)∣<B1​

Next, we'll consider two sequences in RRR to be equivalent iff their difference is null:

σ∼τ⟺null(σ−τ)\sigma \sim \tau \Longleftrightarrow \mathrm{null}(\sigma - \tau)σ∼τ⟺null(σ−τ)

A sequence σ\sigmaσ is null iff

∀B∈N+,∃N∈N:n≥N  ⟹  ∣σ(n)∣<1B\forall B \in \mathbb{N}^+, \exists N \in \mathbb{N} : n \ge N \implies |\sigma(n)| < \frac{1}{B}∀B∈N+,∃N∈N:n≥N⟹∣σ(n)∣<B1​

Now, ∼\sim∼ is an equivalence relation:

  • For any sequence σ\sigmaσ, σ−σ\sigma - \sigmaσ−σ is the constant 000 sequence, which is obviously2 null. So ∼\sim∼ is reflexive.

  • Suppose σ∼τ\sigma \sim \tauσ∼τ and let B∈N+B \in \mathbb{N}^+B∈N+. Then there's some N∈NN \in \mathbb{N}N∈N such that n≥Nn \ge Nn≥N implies ∣σ(n)−τ(n)∣<1B|\sigma(n) - \tau(n)| < \frac{1}{B}∣σ(n)−τ(n)∣<B1​. So for n≥Nn \ge Nn≥N, ∣τ(n)−σ(n)∣<1B|\tau(n) - \sigma(n)| < \frac{1}{B}∣τ(n)−σ(n)∣<B1​ as well. Thus τ∼σ\tau \sim \sigmaτ∼σ, and ∼\sim∼ is symmetric.

  • Finally, suppose ρ σ\rho ~ \sigmaρ σ and σ τ\sigma ~ \tauσ τ and B∈N+B \in \mathbb{N}^+B∈N+. Then there exist N1,N2∈NN_1, N_2 \in \mathbb{N}N1​,N2​∈N such that n≥N1n \ge N_1n≥N1​ implies that ∣ρ(n)−σ(n)∣<12B|\rho(n) - \sigma(n)| < \frac{1}{2B}∣ρ(n)−σ(n)∣<2B1​, and n≥N2n \ge N_2n≥N2​ implies that ∣σ(n)−τ(n)∣<12B|\sigma(n) - \tau(n)| < \frac{1}{2B}∣σ(n)−τ(n)∣<2B1​. Let N=max⁡(N1,N2)N = \max(N_1, N_2)N=max(N1​,N2​). Then n≥Nn \ge Nn≥N implies that

    ∣ρ(n)−τ(n)∣=∣ρ(n)−σ(n)+σ(n)−τ(n)∣≤∣ρ(n)−σ(n)∣+∣σ(n)−τ(n)∣<12B+12B=1B\begin{align*} |\rho(n) - \tau(n)| &= |\rho(n) - \sigma(n) + \sigma(n) - \tau(n)| \\ &\le |\rho(n) - \sigma(n)| + |\sigma(n) - \tau(n)| \\ &< \frac{1}{2B} + \frac{1}{2B} \\ &= \frac{1}{B} \end{align*}∣ρ(n)−τ(n)∣​=∣ρ(n)−σ(n)+σ(n)−τ(n)∣≤∣ρ(n)−σ(n)∣+∣σ(n)−τ(n)∣<2B1​+2B1​=B1​​

    So ρ∼τ\rho \sim \tauρ∼τ, and ∼\sim∼ is transitive.

R\mathbb{R}R as a set of equivalence classes of RRR

We'll define R\mathbb{R}R as the set of equivalence classes of RRR with respect to ∼\sim∼. Symbolically:

R=R /∼\mathbb{R} = R \thinspace / \simR=R/∼

We can think of RRR as the set of names for the real numbers. Note that this means that a single real number may have many distinct names. This is analogous to the situation with the rational numbers, where 12\frac{1}{2}21​ and 24\frac{2}{4}42​ are two different names for the same number.

Equivalence classes are somewhat awkward to work with, so when constructing a number system in this way, we almost always work with the set of names instead. For instance, we'll define addition on RRR, and then show that it can be extended to R\mathbb{R}R because it “respects” ∼\sim∼.

R\mathbb{R}R is a field

Defining +++ and ⋅\cdot⋅ in RRR

Let's first define addition and multiplication in RRR:

σ+τ=n↦σ(n)+τ(n)σ⋅τ=n↦σ(n)⋅τ(n)\begin{align*} \sigma + \tau &= n \mapsto \sigma(n) + \tau(n) \\ \sigma \cdot \tau &= n \mapsto \sigma(n) \cdot \tau(n) \end{align*}σ+τσ⋅τ​=n↦σ(n)+τ(n)=n↦σ(n)⋅τ(n)​

That is, addition and multiplication are defined “pointwise”.

+++ and ⋅\cdot⋅ respect ∼\sim∼

We first need to show that these operations respect ∼\sim∼ in the sense that

σ′∼σ,τ′∼τ  ⟹  σ′+τ′∼σ+τ  ⟹  σ′⋅τ′∼σ⋅τ\begin{align*} \sigma' \sim \sigma, \tau' \sim \tau &\implies \sigma' + \tau' \sim \sigma + \tau \\ &\implies \sigma' \cdot \tau' \sim \sigma \cdot \tau \end{align*}σ′∼σ,τ′∼τ​⟹σ′+τ′∼σ+τ⟹σ′⋅τ′∼σ⋅τ​

This is critical because it means that all of the elements in an equivalence class are treated the same.

Without further ado, suppose that σ′∼σ\sigma' \sim \sigmaσ′∼σ and τ′∼τ\tau' \sim \tauτ′∼τ and B∈N+B \in \mathbb{N}^+B∈N+. Then there exist N1N_1N1​ and N2N_2N2​ such that n≥N1n \ge N_1n≥N1​ implies that ∣σ′(n)−σ(n)∣<12B|\sigma'(n) - \sigma(n)| < \frac{1}{2B}∣σ′(n)−σ(n)∣<2B1​, and n≥N2n \ge N_2n≥N2​ implies that ∣τ′(n)−τ(n)∣<12B|\tau'(n) - \tau(n)| < \frac{1}{2B}∣τ′(n)−τ(n)∣<2B1​. Let N=max⁡(N1,N2)N = \max(N_1, N_2)N=max(N1​,N2​). Then n≥Nn \ge Nn≥N implies that

∣(σ′+τ′)(n)−(σ+τ)(n)∣=∣σ′(n)−σ(n)+τ′(n)−τ(n)∣≤∣σ′(n)−σ(n)∣+∣τ′(n)−τ(n)∣<12B+12B=1B\begin{align*} |(\sigma' + \tau')(n) - (\sigma + \tau)(n)| &= |\sigma'(n) - \sigma(n) + \tau'(n) - \tau(n)| \\ &\le |\sigma'(n) - \sigma(n)| + |\tau'(n) - \tau(n)| \\ &< \frac{1}{2B} + \frac{1}{2B} \\ &= \frac{1}{B} \end{align*}∣(σ′+τ′)(n)−(σ+τ)(n)∣​=∣σ′(n)−σ(n)+τ′(n)−τ(n)∣≤∣σ′(n)−σ(n)∣+∣τ′(n)−τ(n)∣<2B1​+2B1​=B1​​

Showing that multiplication respects ∼\sim∼ is slightly more involved. We'll need to make use of the fact that all sequences in RRR are bounded:

∀σ∈R,∃M∈N:∀n∈N,∣σ(n)∣<M\forall \sigma \in R, \exists M \in \mathbb{N} : \forall n \in \mathbb{N}, |\sigma(n)| < M∀σ∈R,∃M∈N:∀n∈N,∣σ(n)∣<M

And the proof: since σ\sigmaσ is Cauchy, let N∈NN \in \mathbb{N}N∈N such that m,n≥Nm, n \ge Nm,n≥N implies ∣σ(m)−σ(n)∣<1|\sigma(m) - \sigma(n)| < 1∣σ(m)−σ(n)∣<1. Let

μ=max⁡(∣σ(0)∣,∣σ(1)∣,…,∣σ(N−1)∣,∣σ(N)∣)\mu = \max(|\sigma(0)|, |\sigma(1)|, \ldots, |\sigma(N-1)|, |\sigma(N)|)μ=max(∣σ(0)∣,∣σ(1)∣,…,∣σ(N−1)∣,∣σ(N)∣)

and define M=ceil(μ)+1M = \mathrm{ceil}(\mu) + 1M=ceil(μ)+1. Then

n<N  ⟹  ∣σ(n)∣≤μ<M\begin{align*} n < N \implies |\sigma(n)| &\le \mu \\ &< M \\ \end{align*}n<N⟹∣σ(n)∣​≤μ<M​
n≥N  ⟹  ∣σ(n)∣=∣σ(n)−σ(N)+σ(N)∣≤∣σ(n)−σ(N)∣+∣σ(N)∣<1+μ≤M\begin{align*} n \ge N \implies |\sigma(n)| &= |\sigma(n) - \sigma(N) + \sigma(N)| \\ &\le |\sigma(n) - \sigma(N)| + |\sigma(N)| \\ &< 1 + \mu \\ &\le M \end{align*}n≥N⟹∣σ(n)∣​=∣σ(n)−σ(N)+σ(N)∣≤∣σ(n)−σ(N)∣+∣σ(N)∣<1+μ≤M​

So all sequences in RRR are bounded.

Finally we can show that ⋅\cdot⋅ respects ∼\sim∼: given B∈N+B \in \mathbb{N}^+B∈N+, let LLL and MMM be bounds on the size of σ′\sigma'σ′ and τ\tauτ, respectively. Next, let N1,N2∈NN_1, N_2 \in \mathbb{N}N1​,N2​∈N such that n≥N1n \ge N_1n≥N1​ implies that ∣σ′(n)−σ(n)∣<12MB|\sigma'(n) - \sigma(n)| < \frac{1}{2MB}∣σ′(n)−σ(n)∣<2MB1​ and n≥N2n \ge N_2n≥N2​ implies that ∣τ′(n)−τ(n)∣<12LB|\tau'(n) - \tau(n)| < \frac{1}{2LB}∣τ′(n)−τ(n)∣<2LB1​. Let N=max⁡(N1,N2)N = \max(N_1, N_2)N=max(N1​,N2​). Then for n≥Nn \ge Nn≥N,

∣(σ′⋅τ′)(n)−(σ⋅τ)(n)∣=∣σ′(n)τ′(n)−σ(n)τ(n)∣=∣σ′(n)τ′(n)−σ′(n)τ(n)+σ′(n)τ(n)−σ(n)τ(n)∣=∣σ′(n)(τ′(n)−τ(n))+τ(n)(σ′(n)−σ(n))∣≤∣σ′(n)∣∣τ′(n)−τ(n)∣+∣τ(n)∣∣σ′(n)−σ(n)∣<L12LB+M12MB=1B\begin{align*} |(\sigma' \cdot \tau')(n) - (\sigma \cdot \tau)(n)| &= |\sigma'(n)\tau'(n) - \sigma(n)\tau(n)| \\ &= |\sigma'(n)\tau'(n) - \sigma'(n)\tau(n) + \sigma'(n)\tau(n) - \sigma(n)\tau(n)| \\ &= |\sigma'(n)(\tau'(n) - \tau(n)) + \tau(n)(\sigma'(n) - \sigma(n))| \\ &\le |\sigma'(n)||\tau'(n) - \tau(n)| + |\tau(n)||\sigma'(n) - \sigma(n)| \\ &< L\frac{1}{2LB} + M\frac{1}{2MB} \\ &= \frac{1}{B} \end{align*}∣(σ′⋅τ′)(n)−(σ⋅τ)(n)∣​=∣σ′(n)τ′(n)−σ(n)τ(n)∣=∣σ′(n)τ′(n)−σ′(n)τ(n)+σ′(n)τ(n)−σ(n)τ(n)∣=∣σ′(n)(τ′(n)−τ(n))+τ(n)(σ′(n)−σ(n))∣≤∣σ′(n)∣∣τ′(n)−τ(n)∣+∣τ(n)∣∣σ′(n)−σ(n)∣<L2LB1​+M2MB1​=B1​​

We now need to show that +++ and ⋅\cdot⋅ have all of the properties we require of addition and multiplication in a field:

  • σ+τ∼τ+σ\sigma + \tau \sim \tau + \sigmaσ+τ∼τ+σ

  • (ρ+σ)+τ∼ρ+(σ+τ)(\rho + \sigma) + \tau \sim \rho + (\sigma + \tau)(ρ+σ)+τ∼ρ+(σ+τ)
