The Cauchy-Riemann equations

The Cauchy-Riemann equations reveal a shocking contraint on holomorphic functions.

Given a function f:C→Cf: \mathbb{C} \to \mathbb{C}f:C→C, we can define u,v:R2→Ru, v: \mathbb{R}^2 \to \mathbb{R}u,v:R2→R as

Likewise, given u,v:R2→Ru, v : \mathbb{R}^2 \to \mathbb{R}u,v:R2→R, we can define f:C→Cf : \mathbb{C} \to \mathbb{C}f:C→C by

f(x+iy)=u(x,y)+iv(x,y)f(x + iy) = u(x, y) + iv(x, y)f(x+iy)=u(x,y)+iv(x,y)

So every complex function induces a unique1 pair of real scalar-valued functions, and vice-versa. It turns out that fff is holomorphic if and only if
