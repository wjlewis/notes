Newton's method

Newton's method is a basic technique for approximating the roots of a function. The idea is to improve an initial guess using the function's first derivative. Specifically, we construct the line tangent to the function at the guess, r0r_0r0​:

l(x)=f′(r0)x+f(r0)−f′(r0)r0l(x) = f'(r_0)x + f(r_0) - f'(r_0)r_0l(x)=f′(r0​)x+f(r0​)−f′(r0​)r0​

and solve for where it intersects the x-axis:

x=r0−f(r0)f′(r0)x = r_0 - \frac{f(r_0)}{f'(r_0)}x=r0​−f′(r0​)f(r0​)​

We then repeat this process with our (hopefully) improved guess until the difference between successive approximations is within some tolerance. That is

rk+1=rk−f(rk)f′(rk)r_{k+1} = r_k - \frac{f(r_k)}{f'(r_k)}rk+1​=rk​−f′(rk​)f(rk​)​

In python:

def newton_iter(r0, f, df, tolerance=0.1):
    """Approximate a root of f near r0."""
    diff = None
    r = r0
    while True:
        next_r = r - f(r) / df(r)
        diff = abs(r - next_r)
        r = next_r
        if diff < tolerance:
            break

    return r
